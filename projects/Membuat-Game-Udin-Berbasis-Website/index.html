Membuat Game Udin Berbasis Website

HTML,JS,CSS

Jadi Game udin ini adalah projek yang mungkin terbesar ketika saya koding di HP atau Handphone, ketika itu HP saya tidak kuat karena sangking kompleks nya game udin ini, yaa ini bagi H{ ku bagi HP mu mungkin berbeda hehe, lanjut saya mulai membuat sebuah rencana gimana game ini berkerja setelah itu saya membuat documentasinya disini, jadi ini awalnya juga iseng saja membuat game ini karena tidak ada kerjaaan dan yaa melatih javascript yang level lanjutan, ok lanjut kebagian pembuatan

jadi mulainya saya membuat lobi nya dulu di sediakan sebuah 2 tombol yang bertulisan main dan kembali, main untuk melanjutkan main game udin dan kembali untuk tidak bermain
ketika user klik main maka akan di bawakan ke page selanjutnya yaitu menceritakan si udin jadi intinya si udin ini tersesat di dalam hutan dan di menemukan 3 tempat gua
setelah cerita selesai maka disinilah permainan di mulai user harus memilih mana gua yang bener yang harus delewati oleh udin, jikalau salah maka akan berakhir permainnan nya dan kembali di bagian cerita udin

dan lajut ketika user klik kembali di halaman lobi maka akan kembali dari website dan akan ada sound spongebob yaitu tenengggggg gitulahh heheh

kenapa permainan nya kok pendek banget ? karena projek ini di tunda karena saya ada ujian dan masih belum dilanjutkan mungkin nanti yaa tunggu GAME UDIN!!!!!
